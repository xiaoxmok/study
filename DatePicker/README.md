# DatePicker
日期选择插件

#### html例子
```html
<span class="ui-input-date">
            <input type="text" class="datepicker" id="1">
</span>

```

#### js引用
```js
datepicker.init(".datepicker",{
                isClick:false,          //晚于当前时间的日期是否显示
                isTime:true             //是否显示时分秒
            });
```
